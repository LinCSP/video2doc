## 2. Описание компонентов

### 2.1. GUI-слой (wxWidgets)

| Класс | Назначение | Базовый класс wxWidgets |
|-------|------------|-------------------------|
| `MainFrame` | Главное окно приложения, меню, статусная строка | `wxFrame` |
| `SettingsPanel` | Панель настроек: видео, интервал, пути | `wxPanel` |
| `ProjectListPanel` | Управление списком дополнительных проектов | `wxPanel` |
| `PromptEditorPanel` | Редактор шаблона промпта | `wxPanel` |
| `LogPanel` | Панель лога и индикации прогресса | `wxPanel` |
| `ProcessRunner` | Обёртка для запуска внешних процессов | `wxProcess` |

### 2.2. Слой бизнес-логики

| Класс | Назначение |
|-------|------------|
| `TranscriptionEngine` | Управление Этапом 1: подготовка аргументов для ffmpeg и whisper, запуск, мониторинг прогресса. |
| `DocGenerator` | Управление Этапом 2: формирование промпта, запуск Kimi CLI, сбор результатов. |
| `ConfigManager` | Сохранение/загрузка настроек (`wxConfig` или JSON). |
| `PathValidator` | Проверка существования путей, создание директорий. |
| `PromptTemplate` | Работа с шаблоном: подстановка плейсхолдеров, сохранение/загрузка `promt.md`. |

### 2.3. Слой внешних инструментов

| Инструмент | Роль | Интеграция |
|------------|------|------------|
| **ffmpeg** | Извлечение кадров из видео с заданным FPS (`fps=1/INTERVAL`) | Через `wxProcess` + аргументы командной строки |
| **faster_whisper** | Транскрибация речи на русском языке | Через `wxProcess` + Python-скрипт |
| **Kimi Code CLI** | AI-генерация документации на основе промпта и скриншотов | Через `wxProcess` + переменные окружения |

## 4. Управление состояниями

Приложение имеет следующие состояния:

| Состояние | Доступные действия |
|-----------|-------------------|
| `IDLE` | Выбор файлов, редактирование настроек, запуск Этапа 1 или полного цикла. |
| `STAGE_1_RUNNING` | Отображение прогресса ffmpeg/whisper. Доступна кнопка «Отмена». |
| `STAGE_1_DONE` | Результаты Этапа 1 сохранены. Доступен запуск Этапа 2 или повтор Этапа 1. |
| `STAGE_2_RUNNING` | Отображение прогресса Kimi CLI. Доступна кнопка «Отмена». |
| `STAGE_2_DONE` | Документация сгенерирована. Доступен просмотр результатов. |
| `ERROR` | Вывод сообщения об ошибке. Возврат в `IDLE`. |

## 6. Обработка событий wxWidgets

| Событие | Источник | Обработчик |
|---------|----------|------------|
| `wxEVT_BUTTON` (Обзор видео) | `wxButton` | Открыть `wxFileDialog`, обновить путь |
| `wxEVT_BUTTON` (Обзор папки) | `wxButton` | Открыть `wxDirDialog`, обновить путь |
| `wxEVT_BUTTON` (Добавить проект) | `wxButton` | Открыть `wxDirDialog`, добавить в `wxListBox` |
| `wxEVT_BUTTON` (Удалить проект) | `wxButton` | Удалить выбранный элемент из `wxListBox` |
| `wxEVT_BUTTON` (Запуск) | `wxButton` | Запустить `TranscriptionEngine` или `DocGenerator` |
| `wxEVT_END_PROCESS` | `wxProcess` | Обновить состояние, разблокировать кнопки |
| `wxEVT_TIMER` | `wxTimer` | Опрос прогресса ffmpeg (через pipe) |
| `wxEVT_TEXT` | `wxTextCtrl` | Валидация интервала (только цифры) |
