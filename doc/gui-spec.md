## 2. Элементы управления wxWidgets

### 2.1. Главное окно (`MainFrame`)

```cpp
class MainFrame : public wxFrame {
public:
    MainFrame(const wxString& title);
private:
    void CreateMenuBar();      // wxMenuBar
    void CreateStatusBar();    // wxStatusBar
    void LayoutPanels();       // wxBoxSizer + wxNotebook (опционально)
};
```

- **Размер по умолчанию**: 1200×800 px.
- **Минимальный размер**: 900×600 px.
- **Меню**: Файл (Выход), Настройки (Сохранить/Загрузить конфиг), Справка (О программе).

### 2.2. Панель настроек (`SettingsPanel`)

| Элемент | Класс wxWidgets | Свойства |
|---------|----------------|----------|
| Метка «Видео» | `wxStaticText` | — |
| Поле пути к видео | `wxTextCtrl` | `wxTE_READONLY` или редактируемое |
| Кнопка «Обзор...» | `wxButton` | Открывает `wxFileDialog` с фильтром `*.mp4;*.mkv;*.avi` |
| Метка «Интервал» | `wxStaticText` | — |
| Поле интервала | `wxSpinCtrl` | `min=1, max=3600, initial=1` |
| Метка «Выходная папка» | `wxStaticText` | — |
| Поле папки | `wxTextCtrl` | — |
| Кнопка «Обзор...» | `wxButton` | Открывает `wxDirDialog` |

### 2.3. Панель проектов (`ProjectListPanel`)

| Элемент | Класс wxWidgets | Свойства |
|---------|----------------|----------|
| Список проектов | `wxListBox` | `wxLB_SINGLE`, высота ~4-5 строк |
| Кнопка «Добавить...» | `wxButton` | `wxID_ADD`, открывает `wxDirDialog` |
| Кнопка «Удалить» | `wxButton` | `wxID_REMOVE`, удаляет выбранный элемент |

### 2.4. Редактор промпта (`PromptEditorPanel`)

| Элемент | Класс wxWidgets | Свойства |
|---------|----------------|----------|
| Текстовая область | `wxTextCtrl` | `wxTE_MULTILINE \| wxTE_WORDWRAP`, шрифт monospace |
| Кнопка «Сохранить» | `wxButton` | Записывает в `promt.md` |
| Кнопка «Сбросить» | `wxButton` | Возвращает шаблон по умолчанию |

**Шаблон по умолчанию:**

```markdown
Я перевёл видео в текст и картинки {PROJECT_NAME}/*.
Нужно тщательно проанализировать текст и картинки.
И написать документацию в {OUT_DIR}/doc/*.
Также необходимые картинки скопировать в {OUT_DIR}/doc/image/*
и встроить в документацию.
Для документации wxWidgets — mcp Context7.
Если нужно проанализировать сам проект, то {PROJECTS_LIST}.
```

### 2.5. Панель управления и лога (`ControlLogPanel`)

| Элемент | Класс wxWidgets | Свойства |
|---------|----------------|----------|
| Кнопка «Этап 1» | `wxButton` | Запускает `TranscriptionEngine` |
| Кнопка «Этап 2» | `wxButton` | Активна только после успешного Этапа 1 |
| Кнопка «Полный цикл» | `wxButton` | Последовательный запуск 1+2 |
| Кнопка «Отмена» | `wxButton` | `wxID_CANCEL`, вызывает `wxProcess::Kill` |
| Индикатор прогресса | `wxGauge` | `wxGA_HORIZONTAL`, диапазон 0–100 |
| Лог-окно | `wxTextCtrl` | `wxTE_MULTILINE \| wxTE_READONLY \| wxTE_RICH`, моноширинный шрифт |

## 3. Диалоговые окна

### 3.1. Выбор видеофайла (`wxFileDialog`)

```cpp
wxFileDialog openFileDialog(this, _("Выберите видео"), "", "",
    "Видео файлы (*.mp4;*.mkv;*.avi;*.mov)|*.mp4;*.mkv;*.avi;*.mov|Все файлы (*.*)|*.*",
    wxFD_OPEN | wxFD_FILE_MUST_EXIST);
```

### 3.2. Выбор папки (`wxDirDialog`)

```cpp
wxDirDialog dirDialog(this, _("Выберите папку"), defaultPath,
    wxDD_DEFAULT_STYLE | wxDD_DIR_MUST_EXIST);
```

### 3.3. Сообщения об ошибках (`wxMessageBox`)

| Ситуация | Стиль |
|----------|-------|
| Файл не найден | `wxOK \| wxICON_ERROR` |
| Ошибка ffmpeg | `wxOK \| wxICON_ERROR` |
| Этап 1 завершён успешно | `wxOK \| wxICON_INFORMATION` |
| Подтверждение перезаписи | `wxYES_NO \| wxICON_QUESTION` |

## 4. Layout-схема (wxSizer)

```cpp
wxBoxSizer* mainSizer = new wxBoxSizer(wxVERTICAL);

// Верхняя часть: настройки + проекты + промпт (горизонтально)
wxBoxSizer* topSizer = new wxBoxSizer(wxHORIZONTAL);
topSizer->Add(settingsPanel, 1, wxEXPAND | wxALL, 5);
topSizer->Add(projectPanel, 1, wxEXPAND | wxALL, 5);

// Промпт под ними или в отдельной вкладке
mainSizer->Add(topSizer, 0, wxEXPAND);
mainSizer->Add(promptPanel, 1, wxEXPAND | wxALL, 5);

// Кнопки управления
mainSizer->Add(controlPanel, 0, wxEXPAND | wxALL, 5);

// Лог с фиксированной высотой ~200px
mainSizer->Add(logPanel, 0, wxEXPAND | wxALL, 5);

SetSizer(mainSizer);
```

## 5. Тема оформления

- **Шрифт интерфейса**: системный (wxWidgets использует нативные контролы).
- **Шрифт редактора промпта/лога**: `wxFont(wxFontInfo(10).Family(wxFONTFAMILY_TELETYPE))`.
- **Цвета**: системные (wxWidgets обеспечивает нативный look&feel на всех платформах).
- **Иконки**: стандартный набор wxWidgets (`wxArtProvider`) или кастомные 16×16 px.
